# [FR-005] Workflow Loader

## Labels
`MUS`, `takomi`, `config`

## User Story
As a builder, I want the app to read workflow Markdown files so that execution stages follow the VibeCode protocol.

## Proposed Solution

### Overview
Load `.agent/workflows/*.md`, extract workflow metadata, and link workflows to task generation and run guidance.

### Implementation Flow
1. Discover workflow files in the workspace.
2. Parse front matter and body content.
3. Surface workflow references in task prompts and run inspectors.

### Key Considerations
- Workflow parsing must preserve markdown content.
- Workflows need stable keys for task generation.

## Acceptance Criteria

- [ ] Workflow markdown files are loaded at runtime.
- [ ] Workflow names, descriptions, and bodies are queryable.
- [ ] Task generation can attach explicit workflow references.
