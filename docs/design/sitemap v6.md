# Takomi v6 Sitemap

## 0. Initial Entry
- **Project Selector (Clean)**: Entry point to pick or create a workspace.

## 1. Primary Shell
- **Orchestrator Home**: Overview of active sessions, recent work, and system health.
- **Main Session Workspace**: The core "VibeCode" surface where the agent and user collaborate.
  - *Transcript Area*: Streamed text, status blocks, file/folder pills.
  - *Floating Composer*: Advanced prompt box with modes.
  - *Context Inspector*: (Toggleable) Task tree, file explorer, artifacts.

## 2. Management & Configuration
- **Worktree Manager**: Visual git-like branch/worktree switching.
- **Task Tree Monitor**: Deep dive into the current execution graph.
- **Runtime Settings**: System/Agent/Model configuration.
- **Intervention Controls**: Permission gates and manual overrides.

## 3. Account & Verification
- **Billing / Verification**: Usage metrics and account status.
